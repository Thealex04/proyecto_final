# Proyecto de Análisis de Ciclismo

## Descripción
Aplicación en Python para analizar rutas ciclistas personales y competiciones profesionales.

## Tecnologías
- Python
- Pandas
- Matplotlib
- Flask

## Funcionalidades
- Análisis de rutas
- Estadísticas
- Visualización de datos
- Recomendaciones automáticas

## Instalación
pip install -r requirements.txt

## Ejecución
python app.py
